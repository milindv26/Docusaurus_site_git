Perfect Secrecy

Definition​

An encryption scheme (Gen,Enc,Dec)(Gen, Enc, Dec)(Gen,Enc,Dec) with message space M\mathcal{M}M is perfectly secret if for every probability distribution for MMM, every message m∈Mm \in \mathcal{M}m∈M, and every ciphertext c∈Cc \in \mathcal{C}c∈C for which Pr[C=c]>0Pr[C = c] > 0Pr[C=c]>0:

Pr[M=m∣C=c]=Pr[M=m].(1)Pr[M = m | C = c] =Pr[M = m]. \tag{1}Pr[M=m∣C=c]=Pr[M=m].(1)

(The requirement that Pr[C=c]>0Pr[C = c] > 0Pr[C=c]>0 is a technical one needed to prevent conditioning on a zero probability event.)

This definition implies that in a perfectly secret encryption scheme, the ciphertext ccc reveals no information about the plaintext mmm. Specifically, for any message mmm within the message space M\mathcal{M}M and any possible ciphertext ccc in the ciphertext space C\mathcal{C}C, the probability of mmm being the plaintext corresponding to ccc is exactly the same as the probability of mmm being chosen without any knowledge of ccc.

Other notions of Perfect Secrecy​

There are two other notions of perfect secrecy which are articulated through the following two lemmas:

Lemma 1: An encryption scheme (Gen,Enc,Dec)(Gen, Enc, Dec)(Gen,Enc,Dec) with message space M\mathcal{M}M is perfectly secret if and only if the following equation holds for every m,m′∈Mm,m' \in \mathcal{M}m,m′∈M and every c∈Cc \in \mathcal{C}c∈C

Pr[EncK(m)=c]=Pr[EncK(m′)=c].Pr[Enc_K(m) = c] =Pr[Enc_K(m') = c].Pr[EncK​(m)=c]=Pr[EncK​(m′)=c].

Proof:

Part 1: Pr[M=m∣C=c]=Pr[M=m]⟹Pr[EncK(m)=c]=Pr[EncK(m′)=c].Pr[M = m | C = c] =Pr[M = m] \Longrightarrow Pr[Enc_K(m) = c] =Pr[Enc_K(m') = c].Pr[M=m∣C=c]=Pr[M=m]⟹Pr[EncK​(m)=c]=Pr[EncK​(m′)=c].

According to Bayes' theorem

Pr[M=m∣C=c]⏟=Pr[M=m]⋅Pr[C=c]=Pr[C=c∣M=m]⋅Pr[M=m]  ⟹  Pr[C=c]=Pr[C=c∣M=m]  ⟹  Pr[C=c]=Pr[C=c∣M=m′]\begin{align*} \underbrace{Pr[M = m | C = c]}_{=Pr[M = m]} \cdot Pr[C = c] &=Pr[C = c | M = m] \cdot Pr[M = m]\\ \implies Pr[C = c] =Pr[C = c | M = m] \tag{A}\\ \implies Pr[C = c] =Pr[C = c | M = m'] \tag{B}\\ \end{align*}=Pr[M=m]Pr[M=m∣C=c]​​⋅Pr[C=c]⟹Pr[C=c]=Pr[C=c∣M=m]⟹Pr[C=c]=Pr[C=c∣M=m′]​=Pr[C=c∣M=m]⋅Pr[M=m](A)(B)​

Also,

Pr[C=c∣M=m]=Pr[EncK(M)=c∣M=m] by definition,=Pr[EncK(m)=c∣M=m],=Pr[EncK(m)=c] as K is independent of M  ⟹  Pr[C=c∣M=m′]=Pr[EncK(m′)=c]\begin{align*} Pr[C = c | M = m ] &= Pr[Enc_K(M) = c|M=m]\hspace{3px}\text{ by definition},\\ &= Pr[Enc_K(m) = c|M=m],\\ &= Pr[Enc_K(m) = c]\hspace{3px}\text{ as K is independent of M}\tag{C}\\ \implies Pr[C = c | M = m'] &= Pr[Enc_K(m') = c]\tag{D} \end{align*}Pr[C=c∣M=m]⟹Pr[C=c∣M=m′]​=Pr[EncK​(M)=c∣M=m] by definition,=Pr[EncK​(m)=c∣M=m],=Pr[EncK​(m)=c] as K is independent of M=Pr[EncK​(m′)=c]​(C)(D)​

Using (A), (B), (C) and (D), we get

Pr[EncK(m)=c]=Pr[EncK(m′)=c].\begin{align*} Pr[Enc_K(m) = c] =Pr[Enc_K(m') = c]. \end{align*}Pr[EncK​(m)=c]=Pr[EncK​(m′)=c].​

Part 2: Pr[M=m∣C=c]=Pr[M=m]⟸Pr[EncK(m)=c]=Pr[EncK(m′)=c].Pr[M = m | C = c] =Pr[M = m] \Longleftarrow Pr[Enc_K(m) = c] =Pr[Enc_K(m') = c].Pr[M=m∣C=c]=Pr[M=m]⟸Pr[EncK​(m)=c]=Pr[EncK​(m′)=c].

Pr[EncK(m)=c]=Pr[C=c∣M=m],and Pr[EncK(m′)=c]=Pr[C=c∣M=m′] by definition.\begin{align*} Pr[Enc_K(m) = c]&=Pr[C = c | M = m ],\\ \text{and } Pr[Enc_K(m') = c]&=Pr[C = c | M = m']\hspace{3px}\text{ by definition}. \end{align*}Pr[EncK​(m)=c]and Pr[EncK​(m′)=c]​=Pr[C=c∣M=m],=Pr[C=c∣M=m′] by definition.​

Because of the premise, we have

Pr[C=c∣M=m]=Pr[C=c∣M=m′].\begin{align*} Pr[C = c | M = m ]=Pr[C = c | M = m'].\tag{E} \end{align*}Pr[C=c∣M=m]=Pr[C=c∣M=m′].​(E)​

We know that

Pr[C=c]=∑m∈MPr[C=c∣M=m]⋅P[M=m],=Pr[C=c∣M=m1]⋅P[M=m1]+Pr[C=c∣M=m2]⋅P[M=m2]+...+Pr[C=c∣M=m∣M∣]⋅P[M=m∣M∣],=Pr[C=c∣M=m]⋅∑m∈MP[M=m]⏟=1 using (E),=Pr[C=c∣M=m].\begin{align*} Pr[C = c]&=\sum_{m\in \mathcal{M}}Pr[C = c | M = m]\cdot P[M=m],\\ &=Pr[C = c | M = m_1]\cdot P[M=m_1] + Pr[C = c | M = m_2]\cdot P[M=m_2] +...+Pr[C = c | M = m_{|M|}]\cdot P[M=m_{|M|}],\\ &=Pr[C = c | M = m]\cdot\underbrace{\sum_{m\in \mathcal{M}}P[M=m]}_{=1}\hspace{3px}\text{ using (E)},\\ &=Pr[C = c | M = m]. \tag{F} \end{align*}Pr[C=c]​=m∈M∑​Pr[C=c∣M=m]⋅P[M=m],=Pr[C=c∣M=m1​]⋅P[M=m1​]+Pr[C=c∣M=m2​]⋅P[M=m2​]+...+Pr[C=c∣M=m∣M∣​]⋅P[M=m∣M∣​],=Pr[C=c∣M=m]⋅=1m∈M∑​P[M=m]​​ using (E),=Pr[C=c∣M=m].​(F)​

Using Bayes' theorem

Pr[C=c∣M=m]⋅P[M=m]=Pr[M=m∣C=c]⋅P[C=c],  ⟹  P[M=m]=Pr[M=m∣C=c] using (F).\begin{align*} Pr[C = c | M = m]\cdot P[M=m] &= Pr[M = m | C = c]\cdot P[C=c],\\ \implies P[M=m] &= Pr[M = m | C = c]\hspace{3px}\text{ using (F).} \end{align*}Pr[C=c∣M=m]⋅P[M=m]⟹P[M=m]​=Pr[M=m∣C=c]⋅P[C=c],=Pr[M=m∣C=c] using (F).​

■\hspace{480px}\blacksquare■

OTP is Perfectly Secret​

We will use definition (1)(1)(1) to prove that One Time Pad is perfectly secret.

Given:
K=M=C={0,1}l,\mathcal{K}=\mathcal{M}=\mathcal{C}=\{0,1\}^l,K=M=C={0,1}l, where lll is the length of message mmm,

Now we will use Bayes' theorem of conditional probability:

Pr[C=c∣M=m]=Pr[C=c and M=m]Pr[M=m],Pr[M=m∣C=c]=Pr[C=c and M=m]Pr[C=c],  ⟹  Pr[M=m∣C=c]=Pr[C=c∣M=m]⋅Pr[M=m]Pr[C=c],\begin{align*} Pr[C = c | M = m ] &= \frac{Pr[C = c \text{ and } M = m]} {Pr[M = m]},\\ Pr[M = m | C = c ] &= \frac{Pr[C = c \text{ and } M = m]} {Pr[C = c]},\\ \implies Pr[M = m | C = c ] &= \frac{Pr[C = c | M = m ]\cdot Pr[M = m]} {Pr[C = c]},\\ \end{align*}Pr[C=c∣M=m]Pr[M=m∣C=c]⟹Pr[M=m∣C=c]​=Pr[M=m]Pr[C=c and M=m]​,=Pr[C=c]Pr[C=c and M=m]​,=Pr[C=c]Pr[C=c∣M=m]⋅Pr[M=m]​,​

using (2)(2)(2) and (3)(3)(3),

Pr[M=m∣C=c]=Pr[C=c].■\begin{align*} Pr[M = m | C = c ] &= Pr[C = c].\hspace{10px}\blacksquare\\ \end{align*}Pr[M=m∣C=c]​=Pr[C=c].■​

An inherent limitation exists in crafting an encryption scheme that achieves perfect secrecy, which is encapsulated by the following theorem:

Shannon's Theorem for Perfect Secrecy​

Theorem: If (Gen,Enc,Dec)(Gen, Enc, Dec)(Gen,Enc,Dec) is a perfectly secret encryption scheme with message space M\mathcal{M}M and key space K\mathcal{K}K, then ∣K∣≥∣M∣|\mathcal{K}|\geq|\mathcal{M}|∣K∣≥∣M∣.
