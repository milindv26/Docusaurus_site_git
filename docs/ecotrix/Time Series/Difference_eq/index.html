Difference Equations

First Order Difference Equations​

Following equation is the linear first order difference equation:

yt=ϕyt−1+wt.y_t=\phi y_{t-1}+w_t.yt​=ϕyt−1​+wt​.

Recursive Solution​

Consider the following system

DateEquation
0y0=ϕy−1+w0y_0=\phi y_{-1}+w_0y0​=ϕy−1​+w0​
1y1=ϕy0+w1y_1=\phi y_{0}+w_1y1​=ϕy0​+w1​
2y2=ϕy1+w2y_2=\phi y_{1}+w_2y2​=ϕy1​+w2​
⋮\vdots⋮⋮\vdots⋮
tyt=ϕyt−1+wty_t=\phi y_{t-1}+w_tyt​=ϕyt−1​+wt​

Given the initial conditions y−1y_{-1}y−1​ and w0w_0w0​, we are equipped to determine all future values of yyy.

We can write y1y_1y1​ as

y1=ϕy0+w1=ϕ(ϕy−1+w0)+w1=ϕ2y−1+ϕw0+w1.\begin{align*} y_1&=\phi y_{0}+w_1\\ &=\phi (\phi y_{-1}+w_0) +w_1\\ &=\phi^2 y_{-1}+ \phi w_0 +w_1. \end{align*}y1​​=ϕy0​+w1​=ϕ(ϕy−1​+w0​)+w1​=ϕ2y−1​+ϕw0​+w1​.​

Similarly y2y_2y2​ can be written as

y2=ϕ3y−1+ϕ2w0+ϕw1+w2,\begin{align*} y_2&=\phi^3 y_{-1}+ \phi^2 w_0 +\phi w_1 + w_2, \end{align*}y2​​=ϕ3y−1​+ϕ2w0​+ϕw1​+w2​,​

and y100y_{100}y100​ can be written as

y100=ϕ101y−1+ϕ100w0+ϕ99w1+⋯+ϕw99+w100.y_{100}=\phi^{101}y_{-1}+\phi^{100}w_0 + \phi^{99}w_1 + \cdots+\phi w_{99}+w_{100}.y100​=ϕ101y−1​+ϕ100w0​+ϕ99w1​+⋯+ϕw99​+w100​.

In general

yt=ϕt+1y−1+ϕtw0+ϕt−1w1+⋯+ϕwt−1+wt.y_{t}=\phi^{t+1}y_{-1}+\phi^{t}w_0 + \phi^{t-1}w_1 + \cdots+\phi w_{t-1}+w_{t}.yt​=ϕt+1y−1​+ϕtw0​+ϕt−1w1​+⋯+ϕwt−1​+wt​.

Another general form is

yt+j=ϕj+1yt−1+ϕjwt+ϕj−1wt+1+⋯+ϕwt+j−1+wt+j.y_{t+j}=\phi^{j+1}y_{t-1}+\phi^{j}w_t + \phi^{j-1}w_{t+1} + \cdots+\phi w_{t+j-1}+w_{t+j}.yt+j​=ϕj+1yt−1​+ϕjwt​+ϕj−1wt+1​+⋯+ϕwt+j−1​+wt+j​.

Effect of wtw_twt​ on yt+jy_{t+j}yt+j​ is

∂yt+j∂wt=ϕj.\frac{\partial y_{t+j}}{\partial w_t}=\phi^j.∂wt​∂yt+j​​=ϕj.

Points to note:

  1. The above equation is also called Impulse Response Function (IRF).
  2. When ∣ϕ∣<1|\phi|<1∣ϕ∣<1, the system is stable. When ∣ϕ∣>1|\phi|>1∣ϕ∣>1, the system is explosive.
  3. If ∣ϕ∣<1|\phi|<1∣ϕ∣<1, the cumulative effect of one unit increase in wtw_twt​ on yt+jy_{t+j}yt+j​, where j∈{0,1,2,⋯ }j \in \{0,1,2,\cdots\}j∈{0,1,2,⋯}, is
∑j=0∞∂yt+j∂wt=∑j=0∞ϕj=11−ϕ.\sum_{j=0}^{\infty}\frac{\partial y_{t+j}}{\partial w_t}= \sum_{j=0}^{\infty}\phi ^j = \frac{1}{1-\phi}.j=0∑∞​∂wt​∂yt+j​​=j=0∑∞​ϕj=1−ϕ1​.

Example​

Let the stream of future values be {yt,yt+1,yt+2,⋯ }\{y_t,y_{t+1},y_{t+2},\cdots\}{yt​,yt+1​,yt+2​,⋯} and the interest rate be rrr. Present value (PV) of the stream at time ttt is given by

PV=yt+yt+11+r+yt+2(1+r)2+yt+3(1+r)3+⋯PV=y_t+ \frac{y_{t+1}}{1+r}+\frac{y_{t+2}}{(1+r)^2}+\frac{y_{t+3}}{(1+r)^3}+ \cdotsPV=yt​+1+ryt+1​​+(1+r)2yt+2​​+(1+r)3yt+3​​+⋯

Let β\betaβ be the discount factor, hence

β=11+r.\beta=\frac{1}{1+r}.β=1+r1​.

Therefore

PV=yt+βyt+1+β2yt+2+β3yt+3+⋯=∑j=0∞βjyt+j\begin{align*} PV&=y_t+ \beta y_{t+1}+ \beta^2 y_{t+2}+ \beta^3 y_{t+3}+\cdots\\ &=\sum_{j=0}^{\infty}\beta^j y_{t+j}\\ \end{align*}PV​=yt​+βyt+1​+β2yt+2​+β3yt+3​+⋯=j=0∑∞​βjyt+j​​
