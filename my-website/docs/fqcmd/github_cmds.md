# GitHub commands

* Git remote repository url:
```
git remote -v
```
* Yarn depoy to GitHub pages:
```
cmd /C "set GIT_USER=milindv26&& yarn deploy"
```
* Yarn start local server:
```
yarn start
```
* Git merge unrelated histories:
```
git pull --allow-unrelated-histories <repository_name> main
```
