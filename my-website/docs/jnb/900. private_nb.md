# Coding practice

**1. Input is a list of numbers, create a dictionary that contains the frequency of each element in the list**
```python
nums=[1,1,2,3,4,4,4,5,5,2]
freq_dict={}
for i in nums:
    if i in freq_dict:
        freq_dict[i]+=1
    else:
        freq_dict[i]=1
freq_dict
```
Time complexity : **O(n)**
